# 🎵 Gesture Controlled Music Player

Control your music using just hand gestures — no keyboard, no mouse.

Built using **Computer Vision + MediaPipe + OS-level controls**

🤚 Gesture Map
Gesture           Action
----------------------------------
✊              FistPlay / Pause
🖐              Open PalmStop
☝ 1            fingerPrevious Track
✌ 2            fingers (peace)Next Track
🤟 3            fingersVolume Up
🖖 4            fingersVolume Down
👍 Thumbs Up    Like Track ⭐
🤏 Pinch        Mute Toggle

## Tech Stack
- Python
- OpenCV
- MediaPipe
- PyAutoGUI / OS APIs
